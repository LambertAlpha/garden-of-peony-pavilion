# 工程架构

## 技术栈

- **服务器**: Minecraft 1.21.4, 创造模式, 超平坦
- **建造引擎**: Python RCON 脚本 (`uvx --from mcrcon`)
- **辅助**: MCP Server (GardenBot) 用于巡检
- **存档**: `~/minecraft-server/world/`

## RCON 关键参数

| 参数 | 值 | 备注 |
|------|------|------|
| `/fill` 单次上限 | 32,768 方块 | 需要 `safe_fill()` 自动分片 |
| `/clone` 单次上限 | 32,768 方块 | 用于复制重复结构（廊段、柱子） |
| 命令间隔 | 50ms | 低于此值可能崩服 |
| 连接方式 | 长连接，`with MCRcon()` | 不需要反复断开重连 |

## 楼梯方块状态（飞檐核心）

```
minecraft:deepslate_tile_stairs[facing=south,half=bottom]   # 正常放置，低端朝南
minecraft:deepslate_tile_stairs[facing=south,half=top]      # 倒置，做飞檐翘角
minecraft:deepslate_tile_stairs[facing=east,shape=outer_right]  # 外角转角
```

**注意**: `facing=south` 意思是楼梯的低端朝南（人从南边走上去），和直觉可能相反。

## builder.py 核心函数

```python
class MinecraftBuilder:
    def cmd(command)           # 发送单条命令，50ms延迟
    def safe_fill(...)         # 自动分片 fill
    def line(x1,y1,z1,...)    # 3D Bresenham 画线
    def circle(cx,cy,cz,r)    # 中点圆算法（月洞门）
    def clone(src, dest)       # 复制结构
    def clear_area(...)        # fill air，撤销用
```

## 关键工程注意事项

1. **先地形后建筑** — 地形影响所有建筑的 Y 坐标
2. **模板+Clone** — 廊段、柱子做一个模板，批量 clone
3. **每个模块返回 bounding box** — 方便撤销（fill air）
4. **做旧用 `/fill ... replace`** — 随机替换石砖为苔藓石砖
5. **MVP 先行** — 先建一个亭子+小池塘验证全链路

## 超平坦 Y 坐标参考

实测地面 y=-60（草方块），建筑从 y=-59 起：
- 台基：y=-59 ~ -58
- 地面层：y=-57
- 柱子：y=-57 ~ -52（5格高亭）
- 屋顶起始：y=-52
