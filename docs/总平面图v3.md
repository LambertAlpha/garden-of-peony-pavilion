## 一、水面设计（3,070格，28.4%）

### 设计理念
水面整体呈不规则的"如意"形——主湖偏南居中，向西北伸出一条曲水溪流，东北角有假山深潭。三面岸线曲折有湾有岬，北岸以假山堤岸为界，形成"水从山下出"的意象。

### 1.1 主湖（2,702格，占水面88%）

主湖呈不规则形，南北跨度55格（Z=13~68），东西宽度66格（X=26~92），中心偏南。北岸有两个半岛（岬角）伸入水中，南岸有一个浅湾。

```
主湖轮廓（顺时针，26个锚点）:

北岸（复杂，有岬角）:
  P1(30,20) → P2(38,17) → P3(45,21)    [西段北岸，西湾]
  → P4(50,15) → P5(56,13) → P6(62,16)  [北岬1，池馆所在]
  → P7(68,19) → P8(75,16) → P9(80,22)  [北岬2，濯缨水阁所在]
  → P10(86,28)                           [东北角]

东岸:
  P10(86,28) → P11(90,34) → P12(92,42) → P13(90,50)
  → P14(86,56) → P15(82,60)              [东南角]

南岸（有浅湾）:
  P15(82,60) → P16(74,64) → P17(66,67) → P18(58,68)
  → P19(50,67) → P20(42,64) → P21(36,60)

西岸:
  P21(36,60) → P22(32,54) → P23(28,46) → P24(26,38)
  → P25(28,32) → P26(30,26) → P1(30,20) [闭合]
```

关键水面特征:
- **西湾** (P1-P3区域): 浅水，种睡莲，停画船
- **北岬1** (P4-P6): 突出的半岛，上建池馆
- **北岬2** (P7-P9): 较小半岛，上建濯缨水阁
- **南湾** (P18-P21): 最宽阔处，对岸全景最佳观赏点
- **东岸弧线** (P10-P15): 平缓曲线，廊道贴水而行

### 1.2 曲水溪流（约220格）

从主湖西北角 P1(30,20) 向西北延伸，宽3-5格，蜿蜒流向西侧:

```
曲水路径（中心线）:
  S1(30,20) → S2(26,16) → S3(25,15) → S4(20,13)
  → S5(16,16) → S6(14,20) → S7(12,25) → S8(14,30)

溪流宽度: S1-S2宽5格，S2-S3宽5格，S3-S4宽4格
          S4-S5宽3格（最窄，汀步石），S5-S8宽3-4格
末端S8处扩大为翠轩前小池（约10x10不规则形，79格）
```

### 1.3 假山深潭（116格）

位于东北角假山群中:

```

  D1(88,10) → D2(94,8) → D3(100,10) → D4(102,16)

## 四、入口与游线

### 4.1 入口位置: 南墙正中 (58, 90)

理由:
- 南侧是"世俗"一面（闺房/内宅的延伸方向）
- 入口正对远香堂背面，但被影壁挡住，看不到水
- 符合"藏与露"——进门是封闭小院，转弯后才见水面

### 4.2 主游线（杜丽娘游园路线）

```
★入口(58,90)
  ↓ 穿门厅(58,86)
小庭深院(58,80) — "人立小庭深院"
  ↓ 影壁遮挡，左转

  ↓ 沿西岸廊道北上

  ↓ 过九曲桥
曲水溪流(40,20) — "流觞曲水"
  ↓ 石径上山

```

→ 开阔(翠轩望湖) → 曲折(曲水) → 攀升(假山)
→ 梦幻(牡丹亭，全园最高点) → 下降(芍药阑)
→ 水边(池馆) → 跨水(廊桥) → 全景(远香堂) → 收束(回入口)
```
