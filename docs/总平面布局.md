# 总平面布局

> v2 — 基于原文精读 + 建筑规制考证 + 工程可行性审核修订

## 设计原则

1. **半废弃美学** — 花极盛，园半颓。"姹紫嫣红开遍，似这般都付与断井颓垣"
2. **移步换景** — 每次转弯都有新发现，先"藏"后"露"（《园冶》"巧于因借，精在体宜"）
3. **以水为心** — 池塘+流觞曲水，占园面积 1/3+
4. **情绪动线** — 封闭 → 开阔 → 繁华 → 荒芜 → 幽深 → 梦幻
5. **断井颓垣是弥散氛围** — 不是一个集中模块，而是散布在各处的荒废痕迹

## 坐标系统

- **原点** (0, -61, 0): 园林西南角
- **朝向**: X正=东，Z正=南，Y正=上
- **地面**: y=-60（草方块层顶部），建筑从 y=-59 起
- **总范围**: X: 0~80, Z: 0~60（约 80×60 格，≈5亩，合四品知府规制）

## 杜丽娘动线（核心设计依据）

### 《惊梦》游园路线
```
闺房 → 小庭深院 → 画廊(金粉半零星) → 园门 → 
池馆/水边(苍苔) → 满园春花(姹紫嫣红) → 断井颓垣(感慨) → 
翠轩远眺(云霞画船) → 荼蘼花架 → 杜鹃丛 → 
十二亭台(遍赏) → 回闺房 → 入梦 → 
太湖石边 → 芍药栏前 → 牡丹亭畔(梦中相遇)
```

### 《寻梦》重返路线
```
闺房 → 园门(洞开) → 残红满地 → 粉画垣 → 
荼蘼(抓住裙角) → 一湾流水 → 花树边 → 画廊 → 
湖山石边 → 牡丹亭畔 → 雕阑芍药 → 太湖石 → 
雕阑 → 秋千 → 垂杨/榆树 → 
寻遍无人 → 深处发现大梅树一株 → 决意葬此
```

## 建造顺序（修订后）

### Phase 0: 地基标记
- 用临时方块标记关键坐标点（围墙四角、池塘边界、亭位置）
- 确认尺寸和空间感

### Phase 1: 地形与骨架
1. **地形塑造** — 北侧太湖石区堆高（8-10格），南侧平整
2. **池塘开挖** — 35×18，深2-3格，含流觞曲水弯道
3. **围墙** — 白墙 3格+瓦顶 1格=总高4格，含漏窗

### Phase 2: 核心建筑
4. **牡丹亭 F** — 全园最高点，四角攒尖，13格底座
5. **芍药阑 F2** — 紧邻牡丹亭，雕花栏杆围合
6. **翠轩 D** — 临水赏景厅

### Phase 3: 动线系统
7. **入口区 A** — 小庭院+园门+粉画垣
8. **画廊/曲廊** — 连接入口到翠轩到牡丹亭
9. **廊桥** — 跨池塘

### Phase 4: 景观点缀
10. **太湖石 E** — 置石组，留洞可穿行
11. **秋千** — 太湖石与芍药阑之间
12. **荼蘼花架** — 入园路径上
13. **大梅树** — 花园深处偏僻角落（一株！）
14. **花木散植** — 姹紫嫣红遍布
15. **画船** — 停水面
16. **断井+颓垣做旧** — 苔藓石砖随机替换白墙，废井散布
