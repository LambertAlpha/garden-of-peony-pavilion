# 游园惊梦 - Minecraft 庭院重建项目

> "原来姹紫嫣红开遍，似这般都付与断井颓垣。"
> — 汤显祖《牡丹亭·惊梦》

## 项目概述

在 Minecraft 中重建《牡丹亭·游园惊梦》中杜丽娘的后花园。基于原文四折精读 + 明代四品知府园林规制，创造一座**繁华与荒废并存**的梦幻庭院。

## 技术架构

- **服务器**: Minecraft 1.21.4 本地服务器 (`~/minecraft-server/`)
- **建造方式**: Python RCON 脚本（`uvx --from mcrcon`）
- **辅助**: MCP Server (GardenBot) 用于巡检
- **脚本**: `~/minecraft-server/scripts/`
- **存档**: `~/minecraft-server/world/`

## 核心设计理念

1. **牡丹亭+芍药阑** 是不可分割的空间对仗
2. **一株大梅树**（不是梅林）是全剧空间枢纽
3. **断井颓垣是弥散氛围**，不是独立区域
4. **杜丽娘的动线**决定空间序列设计
