# 原文考证

> 基于《牡丹亭》第七出《闺塾》、第九出《肃苑》、第十出《惊梦》、第十二出《寻梦》

## 核心空间线索

### 第七出《闺塾》— 花园初次描述
> 春香：「有**亭台六七座**，**秋千一两架**。绑的**流觞曲水**，面着**太湖山石**。**名花异草**，委实华丽。」

### 第九出《肃苑》— 空间关系
- 从闺阁区到花园有**回廊**连接
- 花园有**花径**需要扫除
- 有**花郎**（园丁）负责维护

### 第十出《惊梦》— 完整游园路线
- 「人立**小庭深院**」— 闺房前的小庭院
- 「**画廊金粉半零星，池馆苍苔一片青**」— 画廊→池馆
- 「**姹紫嫣红开遍**，似这般都付与**断井颓垣**」— 繁花+荒废并存
- 「**朝飞暮卷，云霞翠轩**；**雨丝风片，烟波画船**」— 翠轩+水面画船
- 「便赏遍了**十二亭台**是枉然」— 亭台多座
- 「转过这**芍药栏**前，紧靠着**湖山石**边」— 梦境发生地的精确坐标
- 「将奴搂抱去**牡丹亭畔，芍药阑边**」— 牡丹亭+芍药阑并列

### 第十二出《寻梦》— 最精确的空间定位
- 「**园门洞开**」— 花园有园门
- 「**低就高来粉画垣**」— 粉刷的画墙/照壁
- 「**那一答可是湖山石边，这一答似牡丹亭畔**。嵌**雕阑芍药**芽儿浅，一丝丝**垂杨**线，一丢丢**榆荚**钱」— 空间关系极明确
- 「他倚**太湖石**...捱过**雕阑**，转过**秋千**」— 动线：太湖石→雕阑→秋千
- 「忽然**大梅树一株**，梅子磊磊可爱」— 花园深处一株大梅树
- 「这梅树依依可人，我杜丽娘若死后，**得葬于此**」— 杜丽娘葬身处

## 关键设计洞察

1. **牡丹亭和芍药阑是不可分割的一对** — "牡丹亭畔，芍药阑边"
2. **大梅树是孤独的一株** — 不是梅林，"无人之处"偶然发现
3. **断井颓垣是底色氛围** — 不是角落里的独立区域，是弥散全园的时间痕迹
4. **花园总体基调是"冷静"的** — 老旦语："后花园中冷静"，荒芜中的繁花
