# 材料方案

> v2 — 基于建筑规制考证修订

## 主色板：江南水乡（粉墙黛瓦，漆柱木梁）

| 部位 | 方块 | 方块ID | v1→v2 改动 |
|------|------|--------|-----------|
| **柱子** | ~~去皮云杉~~ → **去皮金合欢原木** | `stripped_acacia_log` | 红棕色，接近漆柱 |
| **梁/斗拱** | 深色橡木 | `dark_oak_log` / `dark_oak_planks` | 不变 |
| **屋顶瓦面** | ~~深色橡木~~ → **深板岩瓦片楼梯+半砖** | `deepslate_tile_stairs` / `deepslate_tile_slab` | 更接近青灰色小青瓦 |
| **白墙** | 白色混凝土 | `white_concrete` | 不变 |
| **墙基** | 石砖或安山岩 | `stone_bricks` / `polished_andesite` | 新增：下部1-2层 |
| **墙顶压瓦** | 深板岩瓦片半砖 | `deepslate_tile_slab` | 新增：黛瓦压顶 |
| **台基** | 石砖+石砖楼梯（2格高） | `stone_bricks` / `stone_brick_stairs` | 加高至2格 |
| **柱础** | 石质半砖或石按钮 | `stone_slab` / `stone_button` | 新增：每柱底部必有 |
| **地面铺装** | 石砖+平滑石交替 | `stone_bricks` / `smooth_stone` | 不变 |
| **小径** | 泥土小径 | `dirt_path` | 曲径通幽 |
| **栏杆** | 金合欢木栅栏 | `acacia_fence` | 改为金合欢配套 |
| **雕阑（芍药阑）** | 金合欢栅栏+活板门组合 | `acacia_fence` + `acacia_trapdoor` | 新增：原文核心 |
| **窗棂** | 丛林木活板门 | `jungle_trapdoor` | 不变 |
| **漏窗** | 铁栏杆排列 | `iron_bars` | 新增：围墙上的镂空 |
| **灯笼** | 灯笼 | `lantern` | 不变 |
| **宝顶** | 避雷针 | `lightning_rod` | 新增：亭顶尖 |
| **门槛** | 半砖横铺 | `stone_brick_slab` | 新增 |

## 园林景观

| 元素 | 方块 | 方块ID | 备注 |
|------|------|--------|------|
| **芍药/牡丹** | 牡丹花（MC中的peony就是芍药） | `peony` | 芍药阑核心植物 |
| **杜鹃** | 杜鹃花丛 | `azalea` / `flowering_azalea` | 映山红 |
| **荼蘼** | 藤蔓+栅栏（花架） | `vine` + `spruce_fence` | 攀援花架 |
| **垂杨** | 橡木树叶+垂藤蔓 | `oak_leaves` + `vine` | 手动搭建垂柳形态 |
| **大梅树** | 樱花树叶（粉色） | `cherry_leaves` + `cherry_log` | 一株！不是林 |
| **榆树** | 橡木原木+橡木树叶 | `oak_log` + `oak_leaves` | 牡丹亭附近 |
| **竹** | 竹子 | `bamboo` | 园角点缀 |
| **太湖石** | 滴水石块+方解石+苔藓块 | `dripstone_block` / `calcite` / `moss_block` | 混用+留洞 |
| **水面** | 水（2-3格深） | `water` | 底部粘土 |
| **睡莲** | 睡莲叶 | `lily_pad` | 池塘 |
| **画船** | 橡木板+红色羊毛顶棚 | `oak_planks` + `red_wool` | 3×8 |
